# Toyota Aqua

This repository contains signal set configurations for the Toyota Aqua, organized by model year and version. The files are structured to allow for easy differentiation between model generations and other vehicle parameters, ensuring accurate signal mapping for each version of the Toyota Aqua.

## Generations

The Toyota Aqua (known as Prius c in international markets) is a subcompact hybrid hatchback that established Toyota's presence in the affordable hybrid segment, spanning two distinct generations:

- **First Generation (2011-2019)**: Launched as Toyota's most affordable hybrid vehicle, the Aqua was introduced at the 2011 Tokyo Motor Show and became one of Japan's most successful vehicle launches. Key features included:
  - Built on enhanced Yaris-inspired platform with compact dimensions
  - Generation 3 Hybrid Synergy Drive system
  - 1.5L DOHC four-cylinder VVT-i gasoline engine (54 kW / 74 PS)
  - Electric motor producing 45 kW (61 PS)
  - Combined system output of 73 kW (100 PS)
  - World-leading fuel efficiency: 35.4 km/L (JC08 cycle), 40.0 km/L (10-15 cycle)
  - Compact dimensions: 3,995mm length, 1,695mm width, 2,550mm wheelbase
  - 260 kg lighter than the standard Prius

- **Market Discontinuation (2019-2020)**: The Prius c was discontinued in international markets as Toyota shifted focus to larger hybrid models:
  - 2019: Final model year in North America
  - 2020: Discontinued in Australia, replaced by XP210-series Yaris Hybrid
  - Replaced in US/Canada by larger E210-series Corolla Hybrid

- **Second Generation (2021-present)**: Japan-exclusive continuation of the Aqua nameplate with significant technological advancement:
  - Unveiled and launched July 19, 2021
  - Built on GA-B platform shared with XP210 Yaris
  - Extended wheelbase by 50mm while maintaining 1,695mm width for compact car classification
  - World's first application of high-output bipolar nickel-metal hydride battery
  - Double the battery output compared to previous generation
  - Enhanced hybrid efficiency and performance

  **Grade Levels:**
  - Launch grades: B, X, G, and Z
  - November 2022: Added GR Sport performance variant
  - Continued focus on Japanese domestic market preferences
